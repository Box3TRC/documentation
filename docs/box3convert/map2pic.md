# 岛三俯视图转换

![1710605133943](map2pic/1710605133943.png)

## 使用步骤
1. 选择地图尺寸
2. 复制生成的代码
3. 打开岛三编辑器，粘贴代码
4. 运行完成后使用<kbd>F12</kbd>Dev tools复制建筑数据并粘贴
5. 点击自动扫描所有方块
    ![1707920469857](blocks2vox/1707920469857.png)
6. 点击列表中的方块修改颜色（修改完可以保存预设到本地）
7. 点击转换
8. 满意后点击`下载png`，否则继续修改。

## 高级设置
无高级设置
